Doom Rooms

Game Description

Solve riddles and puzzles to escape each room! Multiplayer Halloween themed escape rooms. I was mainly involved in writing code and managing player synchronization and interactions for the two rooms in the above video.

My duties included:

- writing scripts for connecting to photon (for multiplayer)

- spawning players

- writing scripts for player interactions

- synchronizing player interactions and movement over the Photon network

- shaders for selectable objects

- level changing (scene fade in/out, player spawn points, etc.)
